微服务名 模块 介绍
1 eurake 注册中心 Spring Cloud Eureka实现
5 zuul 网关 Spring Cloud Zuul实现，路由分发请求到各个微服务
6 sys-service 管理后台 系统基础功能
7 log-center 日志中心 各微服务aop拦截并发送日志，基于rabbitmq处理日志,存储到mysql或者elasticsearch
8 file-center 文件中心 使用FTP文件上传
9 dashboard 监控中心 spring cloud dashboard 实现
说明：

该项目是微服务系统，由Spring Cloud开发，用到Java8、Maven、Mysql、Redis、Rabbitmq

后台使用 spring session，移动端使用前后端分离

权限基于Spring Security注解实现的，因此粒度很细，达到方法级

管理后台内置功能：

菜单管理：菜单列表、添加菜单、修改菜单、删除菜单、菜单图标设置

角色管理：角色查询、添加角色、修改角色、删除角色、分配菜单、分配权限

权限管理：权限查询、添加权限、修改权限、删除权限

用户管理：用户查询、添加用户、修改用户、给用户分配角色、重置密码

文件管理：上传文件、文件列表、文件删除

注册中心：查看微服务注册情况

监控中心：监控微服务信息
